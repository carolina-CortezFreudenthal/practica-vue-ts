# FakeStore

Instalar las dependencias:

```sh
npm install
```

Correr en servidor local

```
npm run dev
```

## Documentación

Esta práctica hace uso de la API de platzi. La documentación puede ser encontrada en [este enlace.](https://fakeapi.platzi.com/en/rest/products)
